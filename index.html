Modified Software

Leopard_SDL2.0.6 - Modified SDL2 version 2.0.6 for PowerPC/Intel Mac OS X Leopard

Leopard_SDL2.0.6 is based on SDL2 version 2.0.6, with one of Thomas Bernard's patches applied and further changes by myself in an attempt to perfect support for Mac OS X Leopard. This SDL version is used to build my SM64EX For Mac OS X Leopard (PPC/Intel) fork when building with SDL2.

SDL and my fork are both licensed under the Zlib license.

Panther_SDL2 - Modified SDL2 version 2.0.3 for Mac OS X 10.3.9, 10.4, & 10.5

Panther_SDL2 is based on SDL2 version 2.0.3, with one of Thomas Bernard's patches as a base, and then further hacked on to get even Mac OS X 10.3.9 to work!

Panther_SDL2 is better supported on Panther, Tiger, and Leopard then Leopard_SDL2.0.6 is, but some programs are now requiring APIs only available in SDL 2.0.4 or even newer (like SM64EX).

The latest FFplay still supports SDL2 version 2.0.3, and both FFplay and Panther_SDL2 are apart of PowerPC Media Center 7.

SDL and my fork are both licensed under the Zlib license.

Parasite Eve Credits Disc Read Fail Glitch Fix

My fix to allow completing the game and enabling 'EX' mode on PSX consoles that may fail to play the ending credit music.

PowerPC Media Center 7 - YouTube/Twitch Streaming, Downloader, Converter, Web Interface, The Latest FFplay, & More For Mac OS X 10.3.9-10.5.8

PowerPC Media Center was in development for years before being discontinued by the original authors 'Hugh Downs' and 'OpenSourceMac' at version 6. When this happened I forked PowerPC Media Center into PowerPC Media Center 7, with a bigger vision, more features, future proofing, and better UX in mind. Since then it has become my most popular project with the most users.

PowerPC Media Center 7 comes with bleeding-edge software depdencies and exciting new features, including an up to date video player with FFplay. Streaming/downloading from your favorite sites such as Twitch, YouTube, and SoundCloud are all still possible in 2021 using this software.

PPCMC7 comes with many dependency programs under thier own respective licenses, but everything else is released into the Public Domain as the former authors did previously with PPCMC version 6.

SM64EX For Mac OS X Leopard (PPC/Intel) - Super Mario 64 Ported To PowerPC Mac OS X Leopard

Turns out I am the first person to release source code modifications that allows the Super Mario 64 port SM64EX to run on Mac OS X Leopard PowerPC and Intel! My fork is not currently in the original SM64EX, but it supports building the master and nightly branches with both SDL 1 and my fork of SDL 2.0.6 named Leopard_SDL2. I have written build instructions to compile yourself.

SM64EX, and by extension my fork do not have any licensing terms.

My Software

21c - BlackJack/21 for every computer

21/BlackJack is my favorite card game. 21c is a text based implementation of it written in C89. There is an 'infinite' deck to draw from, a load/save feature to continue where you left of, a dealer determined to win, the ability to place custom bets, 'graphics', and more!

21C supports anything with a C89 compliant compiler. Releases are available for Mac OS 8+, Mac OS X 10.3.9+, Windows 95+, and MSDOS 3.3+. It has been built by DevCPP 4.99.5, Xcode 3.0, TurboC++ 3.0, and GCC 10.

Build Select - Makes compiling for various Mac OS X versions, SDKs, and arches easy

Mac OS X backwards compatibility used to be pretty great. Set some flags and it usually works itself out. But it has always been a pain to gloabally compile things for older versions.

Build Select changes all that, execute a single command and you'll be compiling for the Mac OS X 10.3.9 SDK using gcc 4.0, for the ppc arch, at the 10.3 API level. You can even set your own custom build flags, if you want to override CFLAGS/LDFLAGS now you simply can.

Build Select is open source software written in bash, licensed under the BSD 3 Clause license.

Dreamcast CDI Burner - Burn all Dreamcast .CDI file types to bootable CD-ROMs on Linux and Mac OS X

The Sega Dreamcast console was released in 1998 and 1999 around the world. It actually had pretty good copy protection mechanisms in place. However, the MIL-CD support in most Sega Dreamcast consoles provided a clever exploit which allows burned CD-ROMs to boot, instead of actual GD-ROM discs. This is in my opinion the best console exploit to ever exist.

Today the most common Sega Dreamcast software file format is .CDI, used by the proprietary Windows only DiscJuggler format. This does not mean you need Windows, because Dreamcast CDI Burner can extract and burn all .CDI Sega Dreamcast file types on Linux and Mac OS X with a collection of open source software.

Open95Keygen - Generate random valid Windows 95/NT 4.0 OEM/Retail Product Keys

Open95Keygen is a portable open source program that generates random valid Product Keys (both OEM and Retail types) for Microsoft's Windows 95 and Windows NT 4.0 operating systems. Mac OS 8/9 68k/PowerPC, Mac OS X 10.3.9-10.6.8 PowerPC, Linux x86_64, and Linux x86 releases are available.

SM64EX is a native Super Mario 64 port derived from the decompilation of Super Mario 64. SM64EXLC is an easy to use, point and click GUI app that compiles my SM64EX fork of the original SM64EX port on PowerPC Mac OS X Leopard into a portable SM64EX.app on your ~/Desktop.

SuperMario64GameSharkWriter - Write GameShark Codes To A USA SM64 ROM file

Super Mario 64 had a lot of cool cheat codes. Now you can easily edit your unmodified or hacked ROM file to always use cheat codes of your choices with the SuperMario64GameSharkWriter program. SM64GSW can either read a list of codes written in a .txt file, or allow you to input codes. It will then edit the ROM for you.

Even better, SM64GSW also patches the checksum check in the ROM file so that emulators can boot the ROM file without further changes. The resulting ROMs work even on Mupen64Plus, which is something the very old CHKSUM64 program can not do. This makwes SM64GSW a better alternative then using CHKSUM64 to get your edited ROMs to boot. You can invoke soley this functionallity using SM64GSW without writing any codes to the ROM as well.

SM64GSW performs messian's patching codes to rom and getting rid of the checksum trap methods entirely in software.

SM64GSW is written in C89 and is licensed under the BSD 3 Clause license. Windows 95-Windows 10, PowerPC Mac OS X, Linux x86, and Linux x86_64 releases are available.
