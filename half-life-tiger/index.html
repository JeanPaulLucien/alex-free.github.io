Xash3D (Half-Life/GoldSrc Engine) For Mac OS X 10.4.x Tiger (Intel)

Ported by Alex Free

Official Homepage | MacintoshGarden

Xash3d is the open source Half-Life/GoldSrc engine by FWGS. Now for the first time ever, an Intel Mac OS X 10.4.x 32 bit build of the latest Xash3d engine is available! Previously when Steam on Mac first released, Half-Life required Mac OS X 10.5.8. This was later made higher, to Mac OS X 10.6.3. Now, Steam requires who knows what to run the client on Mac OS X. This Xash3d build works on Mac OS X 10.4.x (and maybe some other slightly newer Mac OS X versions), and since it's not Steam there is no DRM! You can run mods, play online, and play Half-Life in all it's glory on any Intel Mac.

Usage

You will need a Valve directory from an existing Half-Life installation (I used the GOTY edition of Half-Life). Double click Half-Life.app, and you will be presented with two options:

  • Install Valve Directory - This copies a Valve directory to Half-Life.app/lib/xash3d/valve.
  • Launch Game - This will start the game once a valve directory has been installed.

Install From Source

Requirements

  • Mac OS X 10.4.11 Intel
  • Xcode 2.5
  • MacPorts
  • TigerPorts

Run the build-all script in the source code download. Xash3d will be built into a portable Half-Life.app in the /Applications directory.
